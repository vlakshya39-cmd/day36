# 🧠 Day 36 – Cognitive Pattern Explorer

## 📌 Overview

The **Cognitive Pattern Explorer** is an interactive web application that helps users explore how their thinking patterns change across different situations. Through realistic scenarios, drag-and-drop activities, and self-reflection exercises, the application encourages users to better understand their decision-making style.

This project was built as part of the **#60DaysOfAI Challenge** using AI-assisted development.

---

## ✨ Features

### 🌿 Dual Experience Modes
- Calm Mode
- Stress Mode
- Dynamic visual theme switching

### 🧩 Chapter 1 – Discover Your Thinking Style
- Interactive everyday scenarios
- Multiple-choice decision making
- Personalized thinking analysis

### 🎯 Chapter 2 – Priority Explorer
- Drag-and-drop activity
- Organize personal priorities
- Keyboard accessibility support

### 📊 Chapter 3 – Thinking Pattern Mapping
- Interactive thinking timeline
- Draggable sliders
- Compare logic, emotion, and action preferences

### 📖 Reflection Journal
- Personalized cognitive profile
- Percentage-based thinking breakdown
- Reflection notes
- Educational insights

### ♿ Accessibility
- Keyboard navigation
- Responsive design
- Reduced motion support
- Screen reader friendly elements

## 🚀 How to Run

1. Download the repository.
2. Open `cognitive_pattern_explorer.html`.
3. Launch it in any modern web browser.
4. Choose either **Calm Mode** or **Stress Mode**.
5. Complete all chapters.
6. Review your personalized Reflection Journal.
